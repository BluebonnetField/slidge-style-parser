# Slidge-style parser

A parsing library for [Slidge](https://sr.ht/~nicoco/slidge).
Supports parsing these attributes:

"\_underline_"
"\*bold*"
"~strikethrough~"
"\`code span`"
"\```code block```"
">quote"
"||spoiler||"
"\\\_escape style_"

Most of them corresponds to
[XEP-0393: Message Styling](https://xmpp.org/extensions/xep-0393.html).
